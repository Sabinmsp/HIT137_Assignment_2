This repository contains solutions for HIT137 Assignment 2. The project focuses on implementing Python programs involving encryption/decryption and data processing using file handling and logical operations.

## Question 1 – Encryption & Decryption
This program reads text from a file and applies custom encryption using different shifting rules for letter groups (a–m, n–z, A–M, N–Z) based on `shift1` and `shift2`.  
It then decrypts the text by reversing the process and verifies that the decrypted output matches the original input.
